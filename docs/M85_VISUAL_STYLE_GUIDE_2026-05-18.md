# M85 Visual Style Guide

Date: 2026-05-18

## Target

A small tactical compound at night/rain, closer to a dense playable tactical slice than a clean prototype yard.

## What M85 Adds

- Wet asphalt detail: dark puddle decals and wet-road reflections.
- Route grime: mud scuffs near spawn, warehouse, and building entries.
- Readable checkpoint identity: faded road paint, warning marker, cable bundles.
- Combat residue: small spent-casing clutter near the weapon proof route.
- Building wear: wall grime decals around entry-facing walls.

## What M85 Does Not Claim

M85 does not claim final PUBG-like quality. The current full visual blocker remains:

- first-person hero weapon art still needs final review;
- humanoid art/retarget quality is still intermediate;
- generated GLB class promotion is still incomplete;
- failed Realified non-weapon semantic categories stay quarantined.

## Evidence Rule

Visual polish counts only if it appears in gameplay/player-camera proof and the route gates still pass. Contact sheets and static showcase views are supporting diagnostics only.
