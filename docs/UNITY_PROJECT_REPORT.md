# Unity Project Report

Generated by `AI Tools > Write Project Report`.

## Prototype

- Game: Star Collector Adventure / 星星收集小冒险
- Scene: `Assets/Scenes/Main.unity`
- Goal: move the capsule player, collect 10 stars, and show the win message.

## Main GameObjects

- `Player`: capsule with `CharacterController` and `StarCollectorPlayerController`.
- `Main Camera`: follows the player with `CameraFollow`.
- `Game Manager`: tracks score and win state with `StarCollectorGameManager`.
- `Star 01` ... `Star 10`: trigger collectibles with `CollectibleStar`.
- `Score Text` and `Win Text`: simple UGUI status display.

## How To Play

1. In Unity, choose `AI Tools > Create Star Collector Prototype` if the scene has not been generated yet.
2. Open `Assets/Scenes/Main.unity`.
3. Press Play.
4. Move with WASD or arrow keys.
5. Collect all 10 stars to show `You Win!`.

## Known Limits

- This is a first playable learning prototype using simple geometry, not final art.
- Official Unity MCP is still blocked by Unity plan entitlement; community MCP is configured separately.
